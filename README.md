# Smart Quiz App

A Python-based Quiz Application developed using Tkinter and SQLite.

## Features

- User Login System
- Random Quiz Questions
- Timer for Each Question
- Progress Bar
- Score Calculation
- Leaderboard System
- SQLite Database Integration
- Interactive GUI using Tkinter

## Technologies Used

- Python
- Tkinter
- SQLite3

## How to Run

1. Install Python 3.
2. Download the project files.
3. Open terminal in project folder.
4. Run:

```bash
python quiz.py
```
